## Poolottery!

This app allows you to create lottery pools between friends or co-workers. 

### Options:
  - Purchase tickets through Poolottery
  - Calculate percentages of payouts based on tickets bought
  - Create and join Lottery Pools with random people
  - Be able to create Lottery pools with a cap on members
  - Be able to create a Lottery pool a cap on amount of tickets that can be purchased

Just create a pool and then invite friends to login and join! Purchase tickets and let Poolottery notify you if you've won or not.
